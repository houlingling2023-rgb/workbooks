第 1.1 节　销售分析的重要意义

⏱ 约 25 分钟（认知节，重在想通，不在动手）
🎯先卡你一下

你有个朋友在某 SaaS 公司做售前，月薪不错。他最近天天纠结：要不要辞职，从零开始转"纯数据分析"，去卷 ABtest、用户行为分析？他来问你意见。

你会劝他转，还是劝他别转？凭什么？

先给个判断，再往下看——这节学完，你回答这种问题会很有底气。

🔑这节只攻一个动作

学会两把尺子：① 用"高薪三选一 + toB/toC + 产品复杂度"判断一个销售岗值不值钱；② 用"都是不拿提成的后台岗"判断自己/别人该不该盲目转数据分析。

🧠高手怎么想

一个底层认知：销售才是企业最重要的事

一个企业可以没研发、没产品，什么都可以没有，但不能没有销售——只要能把货卖出去、把钱弄进来，没货可以去进货。打工也一样，最有钱的往往不是程序员，是销售（有的销售总监一年差旅补贴就 50 万）。

尺子一：高薪三选一

一个岗位工资高，原因一定是三选一：

① 技术门槛高     ② 要死很多人才坐得上去（成功率低）     ③ 有权力 / 身份壁垒

销售属于"门槛低、但成功率也低"——所以才"一将功成万骨枯"。

破掉对销售的四个误解

误解真相
销售 = 柜台店员那是最 low 的。高薪销售几乎都是 toB 商务谈判，越大的项目规格越高
销售 = 两条腿跑单/握手线上的买量、投手、商业化、商家运营本质都是销售，只是换了名字
销售 = 靠喝酒越专业的产品越要专业销售；真喝酒的反而是权力垄断型（喝的是"服从"）
互联网不需要销售字节巨量引擎养过十万销售、阿里中供铁军——没销售公司都饿死
🔖 锚点故事·卖 ERP 卖到人民大会堂

陈老师本科同学里第一个发财的，卖 ERP（典型复杂 toB 产品），卖给国家水利部，签约在人民大会堂。2006 年毕业，2008 年就在北京买房了——而你还在拿程序员工资跟人家比。 这告诉你：真正赚钱的销售不是站柜台、不是陪酒，是能拿下大客户的专业商务谈判。

尺子二：该不该转数据分析

商业化 / B 端运营 / 销售运营 / 用户增长 / 买量，和数据分析都是不拿提成的后台岗，等级上去工资一样高。所以：有这些经验的人，没必要丢掉经验从零转数据分析，而该把数据当成"强化工具"，去谋一个更高的运营岗。盲目去卷 ABtest / 用户行为，你没买点经验，是拿短板挤红海。

✍️跟着做一遍（判断三个岗位值不值钱）

用"三选一 + toB/toC + 复杂度"逐个判：

岗位判断
商场化妆品柜台导购toC、门槛低、可被电商/无人店替代 → 不值钱
卖工业 MES 系统的售前/商务toB 复杂产品、要专业谈判、成功率低 → 值钱（命中①技术 +②成功率低）
给国企供货、靠"陪标关系"拿单的销售看着 low，其实是权力/身份壁垒型（命中③）→ 值钱，但靠的是壁垒不是技术
✅ 同样叫"销售"，含金量天差地别——关键看它命中了"三选一"里的哪条。
🪜换你做（二选一 offer）

你之前做过 2 年电商运营，现在拿到两个 offer：

  • A：某大厂"商家运营"（给中小商家做增长，后台岗、不拿提成）
  • B：某创业公司"纯数据分析"，主做用户行为 / ABtest

你选哪个？用本节两把尺子说理由。

👉 参考答案

选 A（商家运营）。理由：

  1. 商家运营 = 商业化/销售类后台岗，和数据分析一样不拿提成、等级上去工资一样高，还能直接用上你 2 年电商运营的业务经验（这是稀缺的"买点经验"）。
  2. B 的纯 ABtest / 用户行为，你恰恰没有买点经验，是从零卷红海——拿短板去拼。
  3. 正确姿势是：进 A，把数据当强化工具来包装工作质量、体现能力，去谋更高的运营岗。

反射：别一听"数据分析"就觉得高级要转过去；先问"我手上的经验在哪个赛道更值钱"。

🃏随身方法卡：30 秒看懂一个销售岗值不值钱
  1. 套三选一：技术门槛高 / 要死很多人才上得去 / 有权力身份壁垒
  2. 看 toB 还是 toC：工资高的几乎都是 toB，项目越大规格越高
  3. 看产品复杂度：越专业/复杂越需要专业销售，跟会不会喝酒无关
  4. 对照自己：有商业化/B端/流量经验，就用数据当强化工具去谋更高运营岗，别盲目转行
⚠️易错＋自测

三个常踩的坑：

  • 还停留在"酒香不怕巷子深"，低估了"说"和表达的价值？
  • 判断岗位/机会，是用"高薪三选一"还是凭感觉？
  • 因为"销售=陪酒/店员"的刻板印象，错过了 toB 商业化这类高价值方向？

自测一题（全新场景）

你表弟说："销售就是站柜台、陪酒喝酒，没技术含量，我才不干。" 用本节"高薪三选一 + 四个误解"反驳他，并举一个高含金量销售岗的例子。

👉 参考解析

反驳四点：

  1. 柜台、陪酒是最 low 的销售；高薪销售是 toB 商务谈判（卖 ERP/MES、卖煤、卖广告）。
  2. 套三选一：高薪岗要么技术门槛高、要么成功率低、要么有壁垒——专业 toB 销售门槛和淘汰率都不低。
  3. 真喝酒的反而是权力垄断型（壁垒③），那是垄断资源、不是没技术。
  4. 互联网全是销售：字节十万销售、阿里中供铁军，只是叫商业化/买量/商家运营。

高含金量例子：卖企业软件的商务、字节巨量引擎的商业化、医药 toB 大客户销售。

第 1.2 节　销售分析的基本思路

⏱ 约 30 分钟
🎯先卡你一下

老板说"这个月销售没完成目标，你分析一下"。你打开系统：有业务员、有商品、有地区、有渠道……几十个维度摆在面前。

你写的第一句结论该是什么？是"A、B 团队都没完成，要加强"吗？

🔑这节只攻一个动作

用"四个核心问题 + 人货厂六点 + 先理销售链路"给一次销售分析定位，并学会"换维度找影响最大的那个、只讲它"。

🧠高手怎么想

四个核心问题（背下来）

① 业绩完成了没有？  ② 谁完成得好/差？  ③ 好的能不能更好？  ④ 差的能不能变好？

业绩一般指销售收入（2B 再加毛利、回款）。任何销售业绩分析，都被这四问框住。

没思路就过"人货厂"六点

  • 人：哪些业务员、哪些顾客不行
  • 货：哪些商品卖得不好，打不打折
  • 厂（渠道）：哪些渠道不行，要不要扩渠道/促销

关键手法：人货厂"交替使用"——换个维度，归因可能完全变

🔖 锚点故事·AB 团队 vs 新旧品

按团队看：A、B 团队都没完成任务，直觉结论"AB 团队都要加强"。但换个维度按新旧品看——发现 AB 团队的旧品都完成了、新品都没完成。问题根本跟团队没关系，是新品卖不动（质量？推广？定价？）。 这告诉你：同一份数据，换个维度归因天差地别。先找影响最大的那个维度，只讲它，别人货厂全堆上。（尤其别学财务写几百页 PPT 却找不到结论。）

数据多少，决定你能分析多深

线下零售（数据最少） < 电话 / 业务员（看数字化程度） < 线上零售（数据最全）

所有销售本质都是转化漏斗，只是数据多少不同。

结果呈现的固定顺序

总业绩完成没 →（线下：按地区拆 / 线上：按漏斗拆——流量·转化率·客单价）

第一步永远是：先梳理本公司销售链路

画流程图 + 标出促销/激励 + 列出报表表头。很多人分析不出来，是因为根本没想清楚自己公司的货是怎么卖的。

✍️跟着做一遍（人货厂交替使用）

总业绩没达标。先按团队看，再换新旧品看：

团队旧品达成率新品达成率
A105%60%
B102%55%
  • 按团队看：A、B 都没完成总任务 → 直觉"AB 团队都要加强"。
  • 换维度（新旧品）：两队旧品都超额（105%/102%）、新品都没完成（60%/55%）→ 问题在货（新品），不在人（团队）。
✅ 该写的结论："新品普遍卖不动（两队新品达成仅 55–60%），与团队无关——去查新品是质量、推广还是定价。" 而不是"AB 团队都要加强"。
🪜换你做（换维度找真因）

某连锁，总业绩没达标。先按地区看，华东、华南都没完成。再给你一个"新客/老客"维度：

地区老客达成率新客达成率
华东110%45%
华南108%40%

问题：直觉结论是什么？换维度后真正的问题在哪？该写什么结论？

👉 对答案
  • 直觉结论："华东、华南两个区都要加强。"（错）
  • 换维度后：两区老客都超额（110%/108%）、新客都崩（45%/40%）→ 问题在拉新质量 / 新客承接，跟地区无关。
  • 该写的结论："新客达成普遍只有 40–45%，是拉新或新客转化出了问题，与区域无关——去查拉新渠道质量和新客承接链路。"

反射：业绩问题别在"地区/团队"上空耗，一定换几个维度（新旧品、新老客、渠道）验证，找到差异最大的那个再下结论。

🃏随身方法卡：给一次销售业绩分析定位的 4 步
  1. 套四问：业绩完成没、谁好谁差、好的能否更好、差的能否变好
  2. 过人货厂：六点逐条筛，标出每个维度的达成率差异
  3. 挑主因：哪个维度差异最大就先讲它，其余先放
  4. 按序呈现：总业绩 →（线下分地区 / 线上拆漏斗：流量·转化率·客单价）
⚠️易错＋自测

三个常踩的坑：

  • 说"人"就甩客户名单、说"货"就甩商品名——有没有打标签、分类？
  • 有没有换维度验证过归因（新旧品 vs 团队、新客 vs 老客）？
  • 清不清楚自己渠道的数字化程度、知道能分析到多深？

自测一题（全新场景）

你是一家线下连锁奶茶的分析师，这个月总销售没达标。老板甩给你一张"各门店达成率"表就要结论。 用本节"四问 + 人货厂 + 先理链路"，说说你动手前会先做什么、可能先换哪几个维度看。

👉 参考解析
  1. 先套四问：总业绩差多少？哪些店好/差？好的店能更好、差的店能变好吗？
  2. 别只在"门店"维度耗，换维度看：是不是某几款新品（货）普遍不行？是不是外卖/堂食某个渠道（厂）掉了？是不是某地区集中下滑？
  3. 线下数据少，先理链路：消费 = 客户数 × 消费率 × 客单价，客户数 = 门前人流 × 进店率。先确认是人流少了（选址/天气/竞品），还是进店/转化掉了（产品/服务/价格）。
  4. 一句话：先梳理销售链路、再换维度定位，最后才下手拆数。

第 1.3 节　销售基本渠道之一：门店

🎯先卡你一下（先别往下看，给自己 1 分钟）

你是一家连锁便利店的数据分析师。老板甩给你两家店的上月数据：

A 店B 店
月收入18 万22 万
房租3 万12 万

老板问："B 店收入更高，是不是该多复制几家 B 这样的店？"

你会怎么回答？ 你手上的数据，够不够回答这个问题？缺什么？

先在心里给个答案，再往下看。这一节就是教你把"凭感觉"变成"算得出、说得清"。

🔑这节只攻一个动作

给一家门店搭一张「单店模型」，再配上「现场标签」，判断它到底行不行、值不值得复制。

只要拿下这一个动作，门店这条渠道你就会分析了。其余都是细节。

🧠高手怎么想

底层规律：位置即流量

零售的第一性原理就一句话——位置即流量。当年李嘉诚被问什么是零售，答案是：location, location, location（位置、位置，还是位置）。

位置一旦选定，客流就基本固定了；之后你能调的，主要是货（陈列、面位、活动）。所以门店分析的优先级是：

选址  >  陈列货架（摆什么、怎么摆）  >  店员水平
🔖 锚点故事·六米生死线

陈老师市区房楼下，一条两车道小马路，两侧各一家早餐店，隔了才六米，卖的也差不多——结果一侧活得好好的，另一侧倒闭了。 为什么？倒闭那侧的楼被酒店包了大半、住的是游客；活下来那侧住的全是住户——住户才是早餐的稳定客流。更要命的是，路中间有栅栏，对面客人过不来，六米宽的小路把客流彻底切成两半。 「六米之差」变成了「生死之差」。（广州整治交通修栅栏，哪修栅栏哪条街的路边店就倒——客人直接不从你这走了。）

这个故事告诉你：决定门店生死的，常常是位置、客群、一道栅栏这些数据库里查不到的现场因素。不去现场，你对着报表永远想不明白这家店为什么不行。

分析框架：左边标签 ✕ 右边模型，两边凑齐才够深

门店分析 = 【左·现场标签】  ✕  【右·单店模型】

左侧（库里通常没有，得去现场打标签）
  位置（路宽 / 公交站远近 / 先前租户 / 相邻铺开关张 / 有无栅栏）
    → 门头装修 → 店员 → 面位·陈列·动线

右侧（库里有，或一问就有）
  固定成本（房租＋水电＋人工）  +  变动成本（进货）
    收入 − 变动成本 = 边际收益
    边际收益 − 固定成本 = 净利润
  • 左边是大多数人缺的：不了解位置、不了解店长水平、不了解装修，就没法解释"它为什么行"。
  • 右边大家也常缺成本：只盯收入，不算房租。普通零售毛利撑死 40%、低的才 25%——一个月租 12 万的店，你得卖到 30 万以上才把租金挣回来，还没算人工。

诊断口诀：先挑标杆（收入高 / 成本低的店）→ 给它打现场标签 → 才解释得清"它为什么行" → 再拿它当尺子去量别的店。

两类决策别混

  • 能不能继续经营：只看每月净利润，正的就能继续开，前期投入先不管。
  • 该不该开新店：净利润之外，还要看前期投入（装修几十万、加盟费上百万）多久回本。
✍️跟着做一遍（A 店标杆，全程示范）

把开场那张表补全，按右侧模型算 A 店（社区便利店，普通零售毛利率取 40%）：

科目数值怎么来的
月收入18 万数据库
变动成本（进货）10.8 万收入 ✕ (1 − 40%) = 18 ✕ 0.6
边际收益7.2 万收入 − 变动成本 = 18 − 10.8
固定成本4.7 万房租 3 ＋ 水电 0.5 ＋ 人工 3人✕0.4 = 1.2
净利润＋2.5 万 / 月边际收益 − 固定成本 = 7.2 − 4.7

  • 位置：临小区正门、两车道无栅栏、住户为主 → 客流天生稳（位置即流量）
  • 门头亮、动线顺、热销品在齐眉货架且面位足 → 转化高
✅ A 店净赚、且现场标签都解释得通 → 它就是标杆店，接下来拿它当尺子。
💡 回到开场那个问题：老板只给你"收入 + 房租"两个数，根本不够——你至少还得拿到毛利率（才能算变动成本）、水电、人工（才能算全固定成本），最后再补现场标签。能不能下结论，先看数据齐不齐。

这正是门店分析最大的坑：只看收入不看成本。

🃏随身方法卡：诊断一家门店的 5 步
  1. 搭单店模型：房租＋水电＋人工（固定）＋ 进货（变动），算出边际收益和净利润
  2. 挑标杆：按"收入高 / 成本低"，选出最赚钱、最不赚钱的店各一家
  3. 去现场打标签：实地看这两家的位置、路宽、门头、动线、面位，记下差异
  4. 对位归因：把差异落到"位置/装修（左侧标签）"还是"成本结构（右侧模型）"
  5. 开新店另算：能不能继续经营看净利润；该不该开新店，还要看前期投入回本速度
⚠️易错＋自测

三个最常踩的坑（自查一遍）：

  • 只看收入、没算成本（尤其房租）——B 店就是这么"看着风光实则亏损"的
  • 位置/装修/店长这些现场标签，是真去看过、还是凭空想的？
  • 用"卖几件 ✕ 单价"硬套，却没解释清楚客流到底从哪来

第 1.4 节　销售基本渠道之二：业务员

⏱ 约 30 分钟
🎯先卡你一下

同样是"卖东西"，下面三个谁离得开业务员、谁离不开？凭什么分？ ① 卖瓶装矿泉水　② 卖二手挖掘机给工地老板　③ 卖一套企业 ERP 系统

先给个判断和理由，再往下看。注意：别用"产品贵不贵"来分。

🔑这节只攻一个动作

拿到一个销售场景，先判断它是简单 2C / 复杂 2C / 还是 2B，再决定怎么分析：复杂 2C 用三色归因，2B 抓客户关系＋预算。

🧠高手怎么想

判据：看"销售过程麻烦不麻烦"，不看产品/生产复杂度

  • 空调：制造很复杂，但你买的那一刻很简单——看参数、网上下单、京东上门售后。→ 简单产品，业务员可被取代。
  • 煤炭：看着简单，其实极复杂——卖给发电/钢铁/化工，要认识人、要分等级、要装车过磅运输。→ 复杂产品，离不开业务员。

结论：判断简单/复杂，看销售过程有多少麻烦事要处理。 简单 2C（瓜子矿泉水）能被无人店/电商取代；复杂 2C（车、房）和 2B（软件、煤）必须业务员。

复杂 2C：把因素分三色，先看能改的

以买车为例（渠道了解→到店接待→推主推品→试驾→选装→金融方案→成交）：

🟢 绿（总部可控）：主推产品比例、优惠力度    ← 订单里看得到，先看这个
🟡 黄（销售个人能力）：接待引导、要不要约试驾、成功案例   ← 销售运营发力处
🟠 橙（用户变量）：用户需求、付费能力        ← 只能初步判断，全程都在影响

2B：先找行业大客户，胜负手是"关系/资源"

  • 一定找固定大客户：发电、钢铁、化工、建材；按规模排：央企 → 国企 → 地方大户 → 散户。
  • 别套 RFM——"消费一个亿、优惠十块"很可笑。要看：客户关系、预算、我占的份额。
🔖 锚点故事·能围标的嫂子

陈老师嫂子是个做国企生意的牛销售，能直接撂话："你想要这个单吗？我知道他们怎么打分，能帮你围下来。" 这告诉你：2B 的胜负手是客户关系和销售资源这类"数据库里没有、要人工打标签"的东西，2C 那套优惠券打法在这儿没用。

业务员渠道一定有的三个结果数：签约收入 / 毛利 / 回款。2B 还要乘一个兑现率（签约≠提货）。

第 1.5 节　销售基本渠道之三：电话销售

⏱ 约 28 分钟
🎯先卡你一下

你接手一个电销团队，接通率只有 1.5%。老板说："优化一下话术，把转化拉一倍。" 你会先做什么？直接去改话术 / AB 测话术，是不是好主意？

先想 1 分钟。

🔑这节只攻一个动作

先判断电销是"盲打"还是"后置"；盲打先想办法把体验前置（变盲打为跟进），再谈精细分析。

🧠高手怎么想

先认清业绩公式和那个死穴

电销业绩 = 坐席数 × 拨打次数 × 接通率 × 有效通话率 × 成交率 × 客单价

死穴是接通率太低（约 1.5%，历史从 10% 一路降到 4% 出头）。接通率这么低时，电话三秒就挂、有效通话不到 10 秒——死磕话术毫无意义，那是时间/人群问题，不是话术问题。盲打模式下，分析基本白费，只能人海战术。

🔖 锚点故事·外呼现场的红灯

外呼现场拉横幅："要底薪是无能的表现，承单才有尊严。" 字节巨量引擎当年的 SMB 团队搞过臭名昭著的"双 100"：每个坐席头上挂盏灯，每天必须呼 100 个电话、有效通话 100 分钟，话筒一放下超时红灯就亮、主管冲过来喊"为什么不打"。 这告诉你：盲打模式下没有分析可言，就是堆人、堆拨打量瞎猫碰死耗子——能离开这种公司就早点离开。

该"大力出奇迹"还是"复制天才"？看销售表现稳不稳

第 1.8 节　销售基本渠道之六：分销

⏱ 约 30 分钟
🎯先卡你一下

🔑这节只攻一个动作

分析经销商，只抓两个核心数：他的库存 + 终端成交价；再用订货行为给经销商分类、区别对待。

🧠高手怎么想

分销 = 借别人的手卖货，层级越多终端越贵

多层代理（总代 → 各级经销商 → 消费者），每多一层就要多让一份利 → 终端价被抬高（格力比美的贵，很大原因是经销层级多）。

经销商忠诚度低，链条会自己崩

趁活动囤货 → 年尾资金紧 → 打折甩货 → 串货 → 价格崩 → 牌子完蛋
🔖 锚点故事·茅台 vs 五粮液（2013）

2013 年塑化剂 + 反腐，白酒集体难过。五粮液一直比茅台卖得好，但这一波没扛住——经销商甩货、品牌崩盘。茅台铁腕控价："谁敢降价甩货，撸掉谁的经销资格"，还逆势扩产防假酒，于是逆势超车成了国酒。 这告诉你：经销商渠道的命门就是库存 + 终端价。管住价格，品牌就稳；管不住，囤货甩货就能把牌子拖垮。

两个核心数，难易不同

  • 终端成交价：相对好查——派业务员/员工去市场调查、抄价。
  • 经销商库存：难——靠自建提货平台（如立白给小商户建提货平台，至少看得到一级大商的库存）。

铁律：经销商 ≠ 消费者

消费者肚皮有限、钱有限；经销商有仓库能囤几年、能融资资金近乎无上限。你对他打折打太狠，他直接融资把货拿光、低价甩到市场上，反把你的价格体系砸了。所以绝不能用 RFM / 优惠券那套对付经销商。

🧠高手怎么想

v1 什么都不知道 → 只能指标拆解：GMV = 用户数 × 转化率 × 客单价
v2 知道有漏斗   → 投放→线索→私聊跟进→下单→发货，分"投放段 / 转化段"看
v3 知道分三次成交 → GMV 拆成 首购 + 二购 + 三购
        首购 = 投放线索 × 转化率 × 客单价
        二/三购 = 上一档存量 × 转化率
解读顺序：先看首购多还是二三购多 → 再看是线索多还是转化率高 → 提假设验证
🔖 锚点故事·"广东转化率高"的两个假设

同样发现"广东转化率高"，可以是两个完全不同的原因：① 广东购买力强、买得起贵的保健品；② 我们的销售和客户都是广东本地人、会讲粤语、和老人好沟通。这是两个不同的业务假设——如果是②，那福建、湖南、四川也是方言区，都可以找本地销售复制。 这告诉你：看到"某指标高"别停，继续追问"为什么高"，落成一条能验证的假设。停在"广东就是高、同行都这么认为"，工资过不了一万五。

🧠高手怎么想

商业底层逻辑

东西能卖出去 = 先有人有需求，然后二选一：

薄利多销（毛利低·量大）：必须把量做起来 → 选址贴客流、省仓储省费用（货直接摆路面）
高毛利：必须多花钱让人觉得"值" → 更好的位置/装修(营销) 或 产品设计(研发) 或 服务(人力)
🔖 锚点故事·两家水果店

A 在高档小区，装修漂亮、车厘子 49.9——高毛利，代价是房租贵、装修贵、启动资金大。B 在城中村，箱子不拆、货摆路面、车厘子 25——薄利多销，代价是必须走量、省掉一切仓储和陈列费用。住高档小区的人，不会开车十几公里去城中村买便宜货。 这告诉你：没有绝对好的打法，只有跟客群匹配的打法。所有高毛利都有代价，薄利多销也有代价——关键是自洽。

2B：一定有固定下游客户

2B 不会独立存在却没有客户。剪映 → 网红/企业小编；酒店用品 → 按房间数、楼层数、前台台数算需求。理解业务 > 套理论：先想清"谁在买、买什么、为什么买"，别张口就要模型。

第 2.3 节　战略层的销售分析

⏱ 约 32 分钟
🎯先卡你一下

你们公司是行业第三，增长比第一名慢，但净利率高一截。老板看着对手的增速眼红，让你分析"要不要也烧钱抢增长"。

你会先比哪几个数？

🔑这节只攻一个动作

拿财报和同行横向对比，确认方向没错——盯三件事：增长是否比对手慢、毛利净利是否比同行低、应收库存有没有雷。别纠结战略层深不深。

🧠高手怎么想
战略层 = 和同行横向对比，确认方向没错（不纠结深不深）
  看三件事：① 增长 vs 对手  ② 毛利/净利 vs 同行  ③ 应收/库存有没有雷
隐形雷：销售额冲高但 应收账款一堆(纸面富贵) / 库存滞销(钱烂仓库)——都不进销售报表
高增长有代价：线上烧推广、线下开店，净利可能很薄（净利率 4% vs 16%）
🔖 锚点故事·洽洽 vs 三只松鼠

洽洽是老牌子、增长慢、丢了老大位；但它自有工厂，毛利反而更好、净利率能到 16%（对手才 4%），而且销售涨、库存反而降。三只松鼠靠烧推广、开直营店冲得很高，结果连跌几年、关了很多店。 这告诉你：战略层不比谁深，比的是"方向对不对"。增长慢但赚得多、底子稳，就不必眼红别人冲得高、摔得惨。

洽洽三只松鼠

🃏随身方法卡：做一次战略层诊断的 4 步
  1. 拉同行：找头部 / 同体量几家的财报
  2. 比三样：增长、毛利净利、应收库存，逐项和同行对位
  3. 判方向：是方向错了，还是只是模式不同（慢但稳）
  4. 给建议：要不要学对手模式，连同风险一起摆给老板定夺

三个常踩的坑：

  • 有没有把公司放到同行里横向比，而不是只看自己？
  • 看销售时，连着看应收和库存了吗？
  • 有没有被"增长快"迷惑，忽略了人家净利其实很薄？

自测一题（全新场景）

你是一家国产美妆品牌（行业第二）的分析师。老板说："对手今年收入涨了 50%，我们才涨 10%，太丢人，必须砸钱追！" 用战略层逻辑回应：你要对比哪几个数、提醒什么风险？

👉 参考解析

要对比：① 双方净利率（美妆毛利普遍高，对手收入涨 50% 很可能净利被营销费砸到很低）；② 应收/库存（对手是不是靠给经销商压货冲收入、库存周转有没有飙）；③ 销售费用率（对手是不是在烧投放/直播）。

提醒：增长有代价——对手可能在烧钱换收入、净利薄、应收库存有雷。我们若 all-in 追，净利会崩，可能像某些网红美妆一样昙花一现。把风险摆给老板定夺，别盲目追。

第 2.4 节　战术层的销售分析

⏱ 约 33 分钟
🎯先卡你一下

给你一张订单表（客户 ID / 商品 / 城市 / 金额 / 折扣），老板问："我们到底靠什么挣钱？"

你会回答"最畅销的是 A001、A002"吗？——这能回答老板吗？

🔑这节只攻一个动作

战术层的灵魂 = 打标签 + 给判断：人先算 RFM 做描述统计、货打"价格带/定位/毛利/折扣"标签、厂打"城市等级/渠道"标签，把冷数字翻译成业务语言。

🧠高手怎么想
人：先算 RFM 做描述统计（看大小客户/会员占比）→ 靠少数大客户(28分) 还是 大量散客(二四分)
    28分 → 服务好金主、拉新重质、产品线丰富
    二四分 → 拉新重量、爆款+搭配、做复购
货：价格带(绝对高中低) / 定位(相对·对标谁) / 毛利率(60%高·40%以下低) / 折扣力度 → 推出"举高打低"等
厂：城市分一二三线、门店分高中低端、线上看推广费/流量/转化
目的：把"A001 卖了 5 件"翻译成"对标国际大牌走亲民路线、举高打低"这种业务语言
🔖 锚点故事·高档餐厅 vs 黄焖鸡

吃一顿上百万的高档餐厅（靠少数大客户·28 分）：厨师菜谱定制、产品线极丰富，一晚伺候一桌就够。黄焖鸡米饭（靠大量散客·二四分）：标品、压成本、一个爆款加几个搭配，靠人多、靠复购。 这告诉你：同是餐饮，标签不同，打法南辕北辙——战术层先用标签认清"靠谁挣钱"，再定拉新/产品线/复购。

✍️跟着做一遍（把货标签推成业务策略）

某品牌一款包：对标 Coach（钱包 700 元），自己卖 500；平时月销很少，但 618 期间销量暴涨；折扣力度大。

打标签 → 推结论：

  • 定位：对标大牌、相对偏低端（500 vs 700）
  • 折扣力度大 + 平时非折扣销量少 + 大促暴涨 → "举高打低"策略（平时装高冷定高价，大促一波把半年任务做掉）
✅ 报告别写"A001 卖得好"，要写"我司对标国际大牌走亲民路线、举高打低"——这才是把数字变业务。
🪜换你做（28 分还是二四分）

某店一年 RFM 描述统计：

指标数值
总客户1000
前 10% 大客户贡献收入占比75%
大客户人均消费是普通客户的 8 倍

问题：这家靠少数大客户还是大量散客？拉新该重质还是重量？产品线该宽还是窄？

👉 对答案
  • 前 10% 贡献 75% + 人均 8 倍 → 典型 28 分，靠少数大客户（金主）。
  • 拉新重质（能不能捞到金主，比数量重要）；产品线宜宽（金主下单多，偶尔做几款不赚钱的让金主买也行）；普通用户活跃不重要，盯紧那 20% 金主在不在。
  • 对照：若"大客户只比普通客户多 1 倍、人均差不多"= 二四分 → 拉新重量、爆款 + 搭配、做复购、产品线别太宽。
🃏随身方法卡：把战术层数据做出业务味的 3 步
  1. 算 RFM 做描述统计：先看大小客户、会员占比，判断 28 分还是二四分
  2. 给货打标签：价格带/定位/毛利/折扣，识别流量款、利润款、举高打低
  3. 给厂打标签：城市等级/线上线下/店位置，把"哪里好卖"翻成可复制的业务结论

三个常踩的坑：

  • 给人、货、厂都打标签了吗，还是在甩清单？
  • 算完 RFM，是先做描述统计，还是急着聚类？
  • 结论是业务语言（亲民路线/举高打低），还是"A001 卖得好"？

第 2.5 节　战斗层的销售分析

⏱ 约 32 分钟
🎯先卡你一下

你管 20 家便利店，想找出"为什么有几家就是卖得特别好"。数据库里只有营业额、租金、SKU 数——没有店内动线、没有店员话术。

你怎么下手？是放弃，还是？

🔑这节只攻一个动作

战斗层缺数据 → 三步：圈标杆（用结果指标挑收入高、成本低的）→ 提可验证假设（可量化、可验证、指向动作）→ 有数据打标签对比、没数据照标杆抄。目标不是 100% 成功，是大概率成功。

🧠高手怎么想
战斗层最大难点：缺数据、细节无法被数据穷尽 → 三步法
  ① 圈标杆：用结果指标(收入/利润/成本)挑出收入高、成本低的 店/投放/业务员
  ② 提假设：为什么它好？提"可量化、可验证、指向动作"的假设
  ③ 标签对比：有数据就打标签对比验证；没数据就定性写学习材料、照标杆抄
圈标杆的结果指标：
  实体店 = 收入/毛利 −（店租+人工+水电+赠品）
  线上推广 = (GMV−退货) × 毛利率 −（推广费+物流+扣点+工资）  ← 退货高时务必先减退货
  业务员 = 签约收入/毛利率/回款  ← 特别看回款，别只考核签约
🔖 锚点故事·小超市的三个假设

小区超市卖不好，三种假设：① "服务不好"——小超市服务没那么重要、个案难量化，扔；② "价格没竞争力"——方向对，但要抄同行价、采集成本高，半好；③ "门前陈列不吸引人"——有商品编号 + 陈列位就能从 POS 流水验证，且直接指向"换陈列"这个动作，好假设。 这告诉你：好假设 = 可量化、可验证、指向一个能改的动作。提"品牌形象不好"这种虚的，不但没法分析，还会把会开成吵架。

超市服务不好❌ 坏个案难量化、小超市服务不是主因
价格没竞争力⚠️ 半好方向对，但抄同行价采集成本高
门前陈列不吸引人✅ 好有商品编号+陈列位可验证，直接指向"换陈列"

👉 对答案

① 别只看营业额选店3（营业额 32 万虽最高，但租金也最高、8 万）。先算"营业额 − 租金"（其他成本相近时的快速排序口径）：店1 = 27 万、店3 = 24 万、店2 ≈ 14.8 万 → 店1 才是标杆（营业额高、租金低）。 ② 可验证好假设举例："店1 的引流爆款（鸡蛋/牛奶）陈列在门口黄金位"——有商品编号 + 陈列位就能从 POS 流水验证，且直接指向"其他店照搬这个陈列"。别提"店1 服务好"（难量化、个案）。

🃏随身方法卡：做战斗层个案诊断的 3 步
  1. 圈标杆：用结果指标挑出收入高 / 成本低的店、投放、业务员
  2. 提好假设：可量化、可验证、指向具体动作（如"换门前陈列"）
  3. 验证或照抄：有数据打标签对比；没数据就定性写学习材料照标杆抄

三个常踩的坑：

  • 有没有先圈标杆，再去找它为什么好？
  • 提的假设可量化、可验证、指向具体动作吗？
  • 有没有被数据形态误导（如在接不通的前提下谈话术）？

自测一题（全新场景）

你管 30 个线上投放计划，想找"为什么有几条素材就是跑得好"。只有素材编号 / 消耗 / 曝光 / 点击 / 转化。 用"圈标杆 → 提可验证假设 → 标签对比"说你怎么做。

👉 参考解析
  1. 圈标杆：算每条转化成本 = 消耗 ÷ 转化，挑成本最低的几条（若有退货先减退货）。
  2. 提可验证假设：别提"这条素材调性好"（没法验证），提"单人物出镜比群像转化高"或"带价格的标题点击率高"——这些能给素材打标签验证。
  3. 标签对比：给 30 条素材打标签（单人/群像、有无价格、有无优惠），对比标杆和差的，找出大概率管用的组合（如"单人物 + 带价格"成本最低）→ 提建议照抄。

目标是大概率成功，不求穷尽每条文案。

第 2.6 节　货 / 场，综合分析的例子：实体店销售

⏱ 约 32 分钟
🎯先卡你一下

一家社区生鲜店开业两个月，营业额一路下滑。开业时送米送油很热闹啊，怎么就掉了？

你会先看哪一类商品的数据？

🔑这节只攻一个动作

🧠高手怎么想
实体店：位置定了，人就基本固定 → 主要分析 货 和 场（陈列/面位/活动/堆头）
社区超市的货分两个角色：
  生鲜（鲜果鲜菜冻品）= 引流款：线上替代不了、卖给社区老人、吸引日常进店
  粮油（米油蛋）     = 首购/硬通货：能囤，适合首次拉新

👉 参考诊断（先自己写完再看）

1. 先扔"店长不上心"——对人不对事，一针对人就变立场之争，没法分析。

2. 不是大环境——龙头蜜雪还 +5%，说明赛道没崩；而鹿见 −18% 比同体量喜茶（−3%）跌得还狠，是自己的问题。谁说"行业饱和"，让他解释蜜雪为什么还在涨。

3. 主因在"货"和"场"：

  • 货：换维度看，经典款各地都稳、新品全线扑街——问题在新品（选品/定价/难喝），不在地区、不在某个人。（这正是 1.2 的"换维度归因"+ 2.4 的"给货打标签"。）
  • 场：外卖渠道占比最大、跌最多——要单独拆外卖漏斗（1.6/1.7 思路：是曝光少、还是进店不下单、还是配送差评/涨价劝退）。

4. 库存隐患：加盟商原料库存周转飙升 + 加盟商抱怨"被压货" = 总部在向渠道压货。注意这里收入是在跌（−18%），所以重点不是"纸面富贵冲收入"（那是收入冲高时的玩法，如 2.3 的三只松鼠），而是 1.8 讲的渠道压货链条：货积在加盟商仓里 → 资金被占、卖不动 → 接下来很可能打折甩货、串货、关店退出 → 把 −18% 拖得更深。也就是说，这 −18% 可能还没见底，压货的雷还没爆。（再叠加 2.3 的提醒：看销售一定连着看库存。）

5. 标杆店A 可复制点：① 经典款占比高、② 不硬推卖不动的新品、③ 外卖运营好（好评率高）。→ 这些正好对应主因，可以照抄。

6. 结构化诊断 + 测试建议：

  • 方向（战略）：赛道没崩、净利率 12% 还行，方向不用大改；但要立刻停止向加盟商压货，否则加盟商甩货/退出会把 −18% 拖成雪崩。
  • 货：暂停"硬压一堆季节限定新品"，回到"经典款为主 + 少量精选新品做测试"；新品先小范围测口味/价格，跑通再铺。
  • 场：单独诊断外卖——拆漏斗看是曝光、转化还是配送/差评/涨价的问题，挑几家店做"外卖优化"测试。
  • 执行（战斗）：把店A 的"经典款陈列 + 外卖运营"做成学习材料，挑 10 家店试点照抄，看营业额能不能回来。
  • 一句话：不是大环境、不是店长不上心，是新品（货）拖累 + 外卖（场）下滑 + 总部压货埋雷；先停压货、收新品、救外卖，用标杆店A 做试点验证。
