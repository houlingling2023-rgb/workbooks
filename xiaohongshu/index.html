🧠高手怎么想

一个前提 + 两套工具：

前提：用户思维（不是自嗨）
  自嗨：以自我为中心 → 发自己想发的
  用户思维：以用户为中心 → 发用户想看的
  核心问题：用户是谁？他们的痛点/渴望是什么？

工具一：爆款内容三大特点（至少命中一个）
  具有实用性 → 帮用户解决实际问题 → 高收藏
  勾起好奇心 → 留悬念/稀缺体验   → 高点击
  引爆情绪点 → 共鸣/爱恨/暖心    → 高点赞+评论

工具二：打造爆款的四个核心要素（缺一不可）
  选题 → 决定有多少人感兴趣
  封面 → 决定用户会不会停下来
  标题 → 决定用户会不会点进去
  正文 → 决定用户会不会互动留下

🧷 记住这个对照：

一位有 7 年运动经验的健身达人来咨询，她在小红书发了不少笔记，播放量都少得可怜。

她发的标题是：《我健身的三个蜕变阶段》《我是什么时候下定决心要加强上肢肌肉力量的》……

被问到"为什么做这些内容"，她说："因为我亲身经历过，比较有经验。"

作者反问她："你观察过小红书上关于健身的爆款笔记在讲什么吗？"

她思考后发现："动作跟练""瘦身瘦腿"方向的视频才是爆款——这命中了实用性，帮用户解决"想减脂/想练好动作"的问题。

她之前发的内容，发在朋友圈也许会有人点赞；但在小红书，没有人认识她，没有人会为自嗨式的表达买单。

有 7 年经验不等于有流量——发用户想看的，才能有流量。

回到 🎯 那条「猫咪晒太阳」视频，用框架过一遍：

第 1 步 · 检验是否有用户思维

问自己：这条视频的目标用户是谁？他们有什么痛点或渴望？

  • 目标用户：养猫的人 + 喜欢看萌宠的人
  • 但这条视频让用户"感觉到跟自己有关"了吗？→ 没有。用户不认识你的猫，不知道看这只猫晒太阳对自己有什么意义。这是典型的自嗨：你觉得猫咪很可爱，但用户需要的不只是"看别人家的猫"，他们需要的是共鸣、信息、或者情绪触动。

第 2 步 · 对照三大特点，检查命中了几个

  • 实用性？❌ 没有干货，没帮用户解决任何问题。
  • 好奇心？❌ "猫咪晒太阳"不稀奇，没留悬念、没反差。
  • 情绪点？⚠️ 勉强算有一点暖感，但画面平淡，情绪刺激不够强烈，很难引爆点赞和评论。

→ 这条视频三大特点都没有强命中，大概率没有爆款潜力。

第 3 步 · 检查四个要素，定位问题出在哪

  • 选题：养猫这个领域受众有，但"猫咪日常"的选题太泛，感兴趣的只是少数；
  • 标题：《今天天气很好，猫咪们享受阳光浴》—— 以"今天"开头，以自我视角叙述，用户看不到"关我什么事"；
  • 封面和正文：内容本身可能不错，但如果前两关（选题/标题）没人点进来，后面再好也没用。

第 4 步 · 如果换成用户思维，这条视频能怎么改？

先问：这批猫咪内容，有没有哪个角度能命中三大特点之一？

→ 可以改成实用性方向：《猫咪总喜欢晒太阳？原来这个行为有这3个含义》—— 帮用户解答一个他们可能好奇的知识点；

→ 或者好奇心方向：《两只猫同住，每天中午发生了这件事》—— 用悬念勾起好奇；

→ 或者情绪点方向（需要剪辑出更有感染力的片段）：《这个下午，我突然理解了为什么养猫的人都说"被治愈了"》

两大内核（先用这把尺评估任何选题）：

爆款选题 = 覆盖人群大 × 痛点程度高

覆盖人群大
  → 对这个话题感兴趣的潜在观众越多，播放量越高
  → 人群太小众 → 几乎注定低流量
  → 判断方法：换一个普通用户站在她的视角问——"我会搜这个吗？"

痛点程度高
  → 选题越直击用户最强烈的需求，点击欲越强
  → 弱痛点（无关紧要）→ 无人在意
  → 强痛点（必须解决/本能焦虑）→ 忍不住点开
  → 判断方法：这个问题"解决了也行、不解决也行"？
    还是"不解决会痛、会损失、会后悔"？
