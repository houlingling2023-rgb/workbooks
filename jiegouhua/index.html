🃏随身方法卡

拆解一个模糊指令的 5 步

  1. 先问起因：这事为什么做、给谁、要达成什么——优先级最高。
  2. 再钉结果：要交付的东西到底长什么样，别让结果含糊。
  3. 连成流程：起因 → 经过 → 结果，整件事连起来想。
  4. 5W2H 补细节：时间 / 地点 / 谁 / 做成啥样 / 多少 / 力度，能自己假设的就别都问。
  5. 删繁就简：用最简方案，只守住"不留明显纰漏"这条底线。
